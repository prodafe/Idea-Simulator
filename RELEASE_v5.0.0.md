## Idea Simulator v5.0.0 — 修订稳定版

**通用场景 AI 推演引擎** — 输入任何场景（求职、创业、投资、搬家、考试、甚至扔垃圾），AI 进行四方推演 + 6维宪法治理校验，输出真实可信的成功率 + 置信区间 + 深度分析报告。

---

### 🧠 场景自适应推演引擎

- **12 场景类型自动识别**：日常/求职/晋升/创业/经营/投资/谈判/搬家/教育/法律/健康/其他
- **LLM 估算基准成功率**：引用真实世界数据（AI岗录取~8%、985录取~1.5%、创业5年存活~15%）
- **自我一致性校验**：乐观+保守双次估算，偏差>15pp自动取均值
- **非商业场景自动适配**：扔垃圾不会套"市场规模"框架
- **LLM 关键因素注入蒙特卡洛**：20000次迭代中应用场景定制化虚拟事件

### 🏛️ 宪法治理层 (Constitutional Governance)

6 维交叉校验确保输出可靠性：

| 校验维度 | 权重 | 职责 |
|---------|:---:|------|
| 真实性锚定 | 22% | 校验成功率/增长率/时间线 vs 真实世界数据 |
| 可行性检查 | 22% | 资金跑道/团队规模/合规障碍 |
| 一致性校验 | 18% | Agent间矛盾检测 |
| 回测对标 | 13% | 186历史案例交叉验证 |
| 幻觉检测 | 13% | 虚构统计/不可验证来源 |
| 社会规范 | 12% | 行为假设/文化适配 |

- 治理分 0-100 → 成功率修正乘数 0.70-1.00
- 违规分级 critical/high/medium/low，报告强制回应严重违规

### 📊 综合现实锚定数据库

12 场景类型 × 50+行业 × 30城市 × 真实统计：
- 求职行业录取率（AI 8%、投行 3%、咨询 2%）
- 教育录取率（985 1.5%、考研 25%、考公 2%）
- 健康恢复率（感冒 98%、手术 97%、癌症早期 90%）
- 投资资产类别（储蓄 2.5%→加密货币 30%风险/收益）
- 10 城市月薪/房租/人才池数据
- 4 国监管时限表
- 186 历史对标案例

### 📝 深度结构化报告

5 段分析框架：核心结论 → 成败拆解(4维度) → 真实世界对比 → 路径推演 → 行动清单。强制引用数据，场景自适应分析维度。

### ⚡ 性能与体验

- LLM 并行化：classify+context、market+risk 并行，省 ~40% 时间
- 取消按钮：AbortController 随时中止推演
- 输入健壮性：拒绝过短/测试词输入并引导
- 置信带 P10-P50-P90 渐变色可视化
- 37 个模型支持（GPT-5.5/Claude Opus 4.7/DeepSeek V4 Pro/GLM-5.1/Qwen3.5 等）

### 🚀 快速开始

```bash
# pip 安装
pip install git+https://github.com/prodafe/Idea-Simulator.git@v5.0.0
idea-simulator

# Docker 部署
docker-compose up -d

# 浏览器打开 http://localhost:8001
```
